== Description ==
A full responsive simple and easy plugin to manage team member with shortcode in page or post.You can display team members picture,social links,and Job Title

Features

1. Grid view to display team members.
2. Limit number of team member to display.
3. Shortcode generator botton in TinyMCE Editor.

__Shortcode__

Don't want to worry about HTML tags?  You can use a shortcode in your posts, pages to dispaly Team Member.

The shortcode to use is `[WPour_team]`.

Here limit  can be send as attributes for limit of the post type team to display in listing.

Backend Editor have TinyMCE button to add shortcode which on click promts and ask to input limit number of post to display and in success, add the shortcode in editor

**Examples**

`[WPour_team limit=6]`


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Use one of the two usage methods (HTML, Shortcode) within the content of your posts or pages.   

4. Use either the HTML or Shortcode methods inside your text widgets.
